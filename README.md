## 사용기술
- FrontEnd: EJS
- BackEnd: Node.js
- Database: PostrgreSQL

## 테스트
1. PostrgreSQL 설치 후 world 데이터베이스 생성
2. 같이 첨부 된 countries.csv, visited_countries.csv 데이터를 world 데이터베이스 추가
3. 패키지 설지
   ```
    npm i
   ```
4. 프로젝트 실행
   ```
    npm run dev
   ```
5. 로컬환경을 통해 접속
   ```
    http://localhost:3000
   ```
6. 나라이름 입력(ex. japan, china, brazil 등)
